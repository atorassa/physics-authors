{\em Resumen.\/} Este trabajo expone una nueva dinámica que puede ser formulada para todos los sistemas de referencia, inerciales y no inerciales.

\bigskip

{\em Keywords:\/} mecánica clásica, dinámica, tensión, fuerza, interacción, masa, aceleración, sistema de referencia inercial, sistema de referencia no inercial.

\normalsize

{\centering\subsection*{Introducción}}

\par Es sabido que la mecánica clásica no puede formular la dinámica de Newton para todos los sistemas de referencia, debido a que ésta no siempre conserva su forma al ser pasada de un sistema de referencia a otro. Si admitimos, por ejemplo, que la dinámica de Newton es válida para un sistema de referencia, entonces no podemos admitir que sea válida para otro sistema de referencia acelerado con respecto al primero, porque el comportamiento de los cuerpos puntuales para el segundo sistema de referencia es distinto a lo establecido por la dinámica de Newton.
\medskip
\par La mecánica clásica soluciona esta dificultad, distinguiendo a los sistemas de referencia: en sistemas de referencia inerciales, para los cuales se formula la dinámica de Newton, y en sistemas de referencia no inerciales, para los cuales no se formula la dinámica de Newton; contradiciendo con esta solución, al principio de relatividad general, que afirma: todos los sistemas de referencia obtendrán las mismas leyes naturales.
\medskip
\par Sin embargo, este trabajo soluciona la dificultad expuesta de la mecánica clásica de una manera diferente, exponiendo una nueva dinámica que podrá ser formulada para todos los sistemas de referencia, inerciales y no inerciales, ya que la misma conservará siempre su forma al ser pasada de un sistema de referencia a otro; o sea, este trabajo expone una nueva dinámica acorde con el principio de relatividad general.

\newpage

{\centering\subsection*{Nueva Dinámica}}

\par Primera definición: La tensión $\vT$ que actúa sobre un cuerpo puntual es una magnitud vectorial y representa un tipo de interacción entre los cuerpos puntuales.
\par La transformación de las tensiones de un sistema de referencia a otro, está dada por la siguiente ecuación:
\begin{eqnarray*}
\vT\rt = \vT
\end{eqnarray*}
\par Segunda definición: La fuerza $\vF$ que actúa sobre un cuerpo puntual es una magnitud vectorial y representa otro tipo de interacción entre los cuerpos puntuales.
\par La transformación de las fuerzas de un sistema de referencia a otro, está dada por la siguiente ecuación:
\begin{eqnarray*}
\vF\rt = \vF
\end{eqnarray*}
\par Tercera definición: La masa $\mM$ que tiene un cuerpo puntual es una magnitud escalar y representa una constante característica del cuerpo puntual.
\par La transformación de las masas de un sistema de referencia a otro, está dada por la siguiente ecuación:
\begin{eqnarray*}
\mM\rt = \mM
\end{eqnarray*}
\par Cuarta definición: La aceleración virtual $\vA^{\circ}$ que tiene un cuerpo puntual es igual a la suma de las fuerzas $\sum \vF$ que actúan sobre el cuerpo puntual dividido por la masa $\mM$ que tiene el cuerpo puntual.
\begin{eqnarray*}
\vA^{\circ} = \frac{\sum \vF}{\mM}
\end{eqnarray*}
\par Primer principio: Un cuerpo puntual puede tener cualquier estado de movimiento.
\par Segundo principio: Las tensiones que actúan sobre un cuerpo puntual A, siempre están en equilibrio.
\begin{eqnarray*}
\sum \vT\ra = 0
\end{eqnarray*}
\par Tercer Principio: Si un cuerpo puntual A ejerce una tensión $\vT$ sobre un cuerpo puntual B, entonces el cuerpo puntual B ejerce una tensión $-\vT$ igual y de sentido contrario sobre el cuerpo puntual A.
\begin{eqnarray*}
\vT\ra = -\vT\rb
\end{eqnarray*}
\par La tensión dinámica $\vT\nD\rab$ ejercida sobre un cuerpo puntual A por otro cuerpo puntual B, causada por la interacción entre el cuerpo puntual A y el cuerpo puntual B, está dada por la siguiente ecuación:
\begin{eqnarray*}
\vT\nD\rab = \vA^{\circ}\ra - \vA^{\circ}\rb
\end{eqnarray*}
\noindent donde $\vA^{\circ}\ra$ es la aceleración virtual del cuerpo puntual A y $\vA^{\circ}\rb$ es la aceleración virtual del cuerpo puntual B.
\par La tensión cinética $\vT\nK\rab$ ejercida sobre un cuerpo puntual A por otro cuerpo puntual B, causada por la interacción entre el cuerpo puntual A y el cuerpo puntual B, está dada por la siguiente ecuación:
\begin{eqnarray*}
\vT\nK\rab = \vA\rb - \vA\ra
\end{eqnarray*}
\noindent donde $\vA\rb$ es la aceleración real del cuerpo puntual B y $\vA\ra$ es la aceleración real del cuerpo puntual A.
\par De los enunciados anteriores, se deduce que la diferencia entre la aceleración virtual $\vA^{\circ}\ra$ y la aceleración real $\vA\ra$ que tiene un cuerpo puntual A es igual a la diferencia entre la aceleración virtual $\vA^{\circ}\rb$ y la aceleración real $\vA\rb$ que tiene otro cuerpo puntual B.
\begin{eqnarray*}
\vA^{\circ}\ra - \vA\ra = \vA^{\circ}\rb - \vA\rb
\end{eqnarray*}

\medskip

{\centering\subsection*{Determinación del Movimiento de los Cuerpos Puntuales}}

\par La ecuación que determina la aceleración real $\vA\ra$ de un cuerpo \hbox {puntual A} con respecto a un sistema de referencia S ligado a un cuerpo puntual S puede ser calculada de la siguiente manera: se tiene por la nueva dinámica que la aceleración virtual $\vA^{\circ}\ra$ y la aceleración real $\vA\ra$ del cuerpo puntual A están relacionadas con la aceleración virtual $\vA^{\circ}\rs$ y la aceleración real $\vA\rs$ del cuerpo puntual S según la siguiente ecuación:
\begin{eqnarray*}
\vA^{\circ}\ra - \vA\ra = \vA^{\circ}\rs - \vA\rs
\end{eqnarray*}
\par Despejando $\vA\ra$ y como la aceleración real $\vA\rs$ del cuerpo puntual S con respecto al sistema de referencia S siempre es igual a cero, entonces se obtiene:
\begin{eqnarray*}
\vA\ra = \vA^{\circ}\ra - \vA^{\circ}\rs
\end{eqnarray*}
\par Por último, reemplazando $\vA^{\circ}\ra$ y $\vA^{\circ}\rs$ por su expresión, o sea utilizando la cuarta definición de la nueva dinámica, entonces queda:
\begin{eqnarray*}
\vA\ra = \frac{\sum \vF\ra}{\mM\ra} - \frac{\sum \vF\rs}{\mM\rs}
\end{eqnarray*}
\par Por lo tanto, la aceleración real $\vA\ra$ de un cuerpo puntual A con respecto a un sistema de referencia S ligado a un cuerpo puntual S estará determinada por la ecuación anterior; donde $\sum \vF\ra$ es la suma de las fuerzas que actúan sobre el cuerpo puntual A, $\mM\ra$ es la masa del cuerpo puntual A, $\sum \vF\rs$ es la suma de las fuerzas que actúan sobre el cuerpo puntual S y $\mM\rs$ es la masa del cuerpo puntual S.

\medskip

{\centering\subsection*{Conclusiones}}

\par A diferencia de lo que establece la primera y segunda ley de Newton, de la última ecuación se deduce que la aceleración real del cuerpo pun- tual A con respecto al sistema de referencia S no depende solamente de las fuerzas que actúan sobre el cuerpo puntual A sino que también depende de las fuerzas que actúan sobre el cuerpo puntual S donde se halla ligado el sistema de referencia S. O sea, para el sistema de referencia S el cuerpo puntual A puede tener una aceleración real aún si sobre el cuerpo pun- tual A no actúa fuerza alguna y también para el sistema de referencia S el cuerpo puntual A puede no tener una aceleración real (estado de reposo o de movimiento rectilíneo uniforme) aún si sobre el cuerpo puntual A actúa una fuerza no equilibrada.
\medskip
\par Por otro lado, de la última ecuación también se deduce que la segunda ley Newton será válida para el sistema de referencia S solamente si la suma de las fuerzas que actúan sobre el cuerpo puntual S es igual a cero. Por lo tanto, el sistema de referencia S será un sistema de referencia inercial si la suma de las fuerzas que actúan sobre el cuerpo puntual S es igual a cero pero será un sistema de referencia no inercial si la suma de las fuerzas que actúan sobre el cuerpo puntual S no es igual a cero.
\medskip
\par Además, se notará que por medio de la nueva dinámica un sistema de referencia no inercial podrá describir el comportamiento (movimiento) de un cuerpo puntual exactamente de la misma forma que lo hace un sistema referencia inercial y sin necesidad de introducir fuerzas ficticias (conocidas como pseudofuerzas, fuerzas inerciales o fuerzas no inerciales).
\medskip
\par Por último, se notará también que la nueva dinámica seguiría siendo válida aún si la tercera ley de Newton no fuese válida.

{\centering\subsection*{Comportamiento Dinámico de los Cuerpos Puntuales}}

\vspace{-0.9mm}

\par Según la dinámica de Newton el comportamiento de dos cuerpos puntuales A y B estará determinado para un sistema de referencia S (inercial) por las siguientes ecuaciones:
\begin{eqnarray}
\sum \vF\ra = \mM\ra\vA\ra \ \ \ \ \ \ y \ \ \ \ \ \ 
\sum \vF\rb = \mM\rb\vA\rb
\end{eqnarray}
\noindent o sea:
\begin{eqnarray}
\frac{\sum \vF\ra}{\mM\ra} - \vA\ra = 0 \ \ \ \ \ \ y \ \ \ \ \ \ 
\frac{\sum \vF\rb}{\mM\rb} - \vA\rb = 0
\end{eqnarray}
\par Igualando las ecuaciones (2), se obtiene:
\begin{eqnarray}
\frac{\sum \vF\ra}{\mM\ra} - \vA\ra = \frac{\sum \vF\rb}{\mM\rb} - \vA\rb
\end{eqnarray}
\par Por lo tanto, el comportamiento de los cuerpos puntuales A y B estará determinado ahora para el sistema de referencia S por la ecuación (3).
\par Si se pasa la ecuación (3) del sistema de referencia S a otro sistema de referencia S' (inercial o no inercial), utilizando la transformación de la cinemática: $\vA\rt = \vA - \vA\rot$ y las transformaciones de la dinámica: $\vF\rt = \vF$ y $\mM\rt = \mM$, se deduce:
\begin{eqnarray}
\frac{\sum \vF\rt\ra}{\mM\rt\ra} - \vA\rt\ra = \frac{\sum \vF\rt\rb}{\mM\rt\rb} - \vA\rt\rb
\end{eqnarray}
\par Ahora, como la ecuación (4) tiene la misma forma que la ecuación (3) entonces se puede establecer que el comportamiento de los cuerpos puntuales A y B estará determinado por la ecuación (3) para cualquier sistema de referencia (inercial o no inercial); siendo la ecuación (3) igual a:
\begin{eqnarray}
(\frac{\sum \vF\ra}{\mM\ra} - \frac{\sum \vF\rb}{\mM\rb}) + (\vA\rb - \vA\ra) = 0
\end{eqnarray}
\noindent y también igual a:
\begin{eqnarray}
(\frac{\sum \vF\rb}{\mM\rb} - \frac{\sum \vF\ra}{\mM\ra}) + (\vA\ra - \vA\rb) = 0
\end{eqnarray}
\par Siendo finalmente las ecuaciones (5) y (6) las ecuaciones básicas para el desarrollo de la nueva dinámica.
