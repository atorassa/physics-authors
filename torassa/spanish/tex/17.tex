{\bf Resumen}

\bigskip

\parbox{90mm}{En este trabajo una ecuación para calcular el momento de inercia de un sistema de bipartículas es presentada.}

\end{center}

\normalsize

\bigskip \medskip

\noindent \hspace{+0.6em} El momento de inercia de un sistema de partículas, está dado por:
\begin{eqnarray*}
\EI\ri^{\vphantom{\dos}} = \sum_{\scriptscriptstyle i} \; \mM\ri^{\vphantom{\dos}} \, \vR\ri\dos
\end{eqnarray*}

\noindent \hspace{+0.6em} El momento de inercia de un sistema de bipartículas, está dado por:
\begin{eqnarray*}
\EI\rij^{\vphantom{\dos}} = \sum_{\scriptscriptstyle i} \, \sum_{\scriptscriptstyle j>i} \; \mM\ri^{\vphantom{\dos}} \, \mM\rj^{\vphantom{\dos}} \left( \vR\ri - \vR\rj \right)^2
\end{eqnarray*}

\noindent \hspace{+0.6em} Un sistema de partículas forma un sistema de bipartículas, y desde las ecuaciones anteriores se puede obtener la siguiente relación:
\begin{eqnarray*}
\EI\rij^{\vphantom{\ccc}} = \EM\ri^{\vphantom{\ccc}} \: \EI\ri\ccc
\end{eqnarray*}

\noindent donde $\EI\rij^{\vphantom{\ccc}}$ es el momento de inercia del sistema de bipartículas, $\EM\ri^{\vphantom{\ccc}}$ es la masa del sistema de partículas y $\EI\ri\ccc$ es el momento de inercia del sistema de partículas con respecto al centro de masa.
